## 📖 使用说明

### 何时查阅修复记录？

1. **遇到类似问题**: 查看是否有相关的历史解决方案
2. **了解问题演进**: 理解某个功能的问题修复历程
3. **学习调试技巧**: 从历史问题中学习调试和修复方法

### 问题分类

- **数据库**: 表结构、索引、查询相关
- **服务**: 嵌入、生成等后端服务
- **认证**: JWT、OAuth2 等认证问题
- **前端**: Vue 组件、状态管理等
